# Minor Cybersecurity
Deze repository bevat alle opdrachten, aantekeningen en uitwerkingen van mijn Associate Degree Minor Cybersecurity.  
De inhoud is per vak opgesplitst in folders. In elke folder staat het werk dat ik tijdens blok 3 voor dat vak heb uitgevoerd, inclusief de bestanden die zijn ingeleverd via Portflow.

## Vakken
[Operational Technologies](https://github.com/vsheo/Minor-AD-CS-blok-3/tree/main/OT)
Bevat Markdown-bestanden van de opdrachten, inclusief bronverwijzingen die mij hebben geholpen en screenshots van mijn bevindingen tijdens het uitvoeren van de opdrachten.

[Netwerken](https://github.com/vsheo/Minor-AD-CS-blok-3/tree/main/netwerken)
Hier staan mijn samenvattingen over netwerken, gebaseerd op de Packet Tracer-video’s en bijbehorende opdrachten.

[Programmeren](https://github.com/vsheo/Minor-AD-CS-blok-3/tree/main/pythonTelegramBot)
Deze map bevat de code van mijn Python Telegram-bot. De code is opgesplitst volgens het Atomic Design-principe.  
In de issues met het label programmeren heb ik het ontwikkelproces en de gemaakte functies bijgehouden.

[Risicomanagement](https://github.com/vsheo/Minor-AD-CS-blok-3/tree/main/risicomanagement)
Bevat PDF bestanden van de uitgewerkte opdrachten voor risicomanagement, evenals tussentijdse Excel bestanden die via Portflow moesten worden ingeleverd.

[Road to Young Professional](https://github.com/vsheo/Minor-AD-CS-blok-3/tree/main/road-to-young-profesional)
Hier staan de PDF bestanden van alle opdrachten voor het vak Road to Young Professional, inclusief de Weekly Hack presentaties die in submappen zijn georganiseerd.
